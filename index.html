Ground Out
GO is a single page application built with AngularJS, Firebase, and Bootstrap. It's targeted toward baseball lovers who are interested in seeing every major league ballpark in the country. It allows users to track their progress as they travel, make comments on each park, view comments of other users, and browse the cheapest tickets at each park via the SeatGeek api. Check it out here or the code on Github
Vinyl Woof
Vinyl Woof is a single page app I am currently building with ASP.NET MVC, AngularJS, Entity Framework, and Bootstrap. Inspired by the site PaperBackSwap, VW is a place where users can give away their old vinyl records to redeem credits (called "Woofie", an homage to the novel, Down And Out In The Magic Kingdom) which they can then use to claim albums from other users. Check it out here or view the Github repository
Gin Mixer
Gin Mixer is a WPF application built with .NET and SQL Express. It's a windows desktop app where users can select cocktail ingredients they have on hand and check which recipes they can make from the database. If you've got a favorite gin cocktail that's not already in the database, just add new drink to it and its recipe and ingredients will be added to the list. Check it out on Github
